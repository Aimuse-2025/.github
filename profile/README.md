# Aimuse 🎹

## AI가 연주를 악보로, 악보를 연습으로 🎼

  > AI를 통해 누구나 쉽게 악기 연주를 악보로 만들고, 체계적인 연습을 즐길 수 있도록 돕습니다.

<br>

### 주요 기능

#### ✅ AI 기반 자동 악보 생성
좋아하는 MP3 파일을 업로드하면 Your MT3 모델이 자동으로 음원을 분석하여 고품질 PDF 악보를 생성해요. <br>
복잡한 라이브 음원이나 소음이 포함된 음원도 정확하게 변환할 수 있어요.

#### ✅ 무제한 곡 선택의 자유
제한된 곡 라이브러리가 아닌, 사용자가 원하는 모든 MP3 파일을 업로드하여 연습할 수 있어요. <br>
K-POP, 클래식, 인디 음악 등 장르 제한 없이 나만의 연습곡을 만들어 보세요.

#### ✅ 실시간 피드백으로 효과적인 연습
Web MIDI API 기반의 실시간 판정 시스템으로 연주 실력을 즉시 확인할 수 있어요. <br>
USB 피아노를 연결하면 PERFECT/GREAT/GOOD/MISS 판정과 함께 점수, 콤보, 정확도가 실시간으로 표시되어요.

#### ✅ Simply Piano 스타일의 직관적인 연습 화면
Canvas 기반의 부드러운 노트 롤 애니메이션으로 음표가 흐르는 것을 시각적으로 확인할 수 있어요. <br>
반주와 함께 연습하며, 타겟 라인에 맞춰 건반을 누르면 즉각적인 피드백을 받을 수 있어요.

### 서비스 차별점 및 장점

#### 👌 복잡한 음원도 정확하게
라이브 음원이나 관객 소음이 포함된 복잡한 음원도 음원 분리 및 노이즈 제거 기술로 고품질 악보 생성 가능

#### 👌 무제한 음원 자유
제한된 곡 라이브러리가 아닌, 사용자가 원하는 모든 MP3 파일을 업로드하여 자유롭게 연습 가능

#### 👌 올인원 통합 플랫폼
악보 생성부터 실시간 피드백 연습까지 하나의 서비스에서 끊김 없이 제공하는 업계 최초 통합 경험
